## KeyBindings
    
    SUPER + W → Close active window
    SUPER + M → Exit session
    SUPER + R → Open app launcher
    SUPER + L → Lock screen
    SUPER + 1-9 → Switch to workspace 1-9
    SUPER + Shift + 1-9 → Move active window to workspace 1-9
    SUPER + RETURN → Open terminal emulator
